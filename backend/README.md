# Cat API (FastAPI)

Simple API for managing users, cats, documents, events, entities, and locations.

## Requirements

- Python 3.13+
- FastAPI
- Uvicorn

## Run

Navigate into the backend folder:

```bash
cd backend
```

Start the uvicorn application:

```bash
uvicorn main:app --reload
```
