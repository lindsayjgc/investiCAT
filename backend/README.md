# Cat API (FastAPI)

Simple API for managing users, cats, documents, events, entities, and locations.

## Requirements

- Python 3.13+
- FastAPI
- Uvicorn

## Run

From the root of the project:

```bash
uvicorn backend.main:app --reload
```
